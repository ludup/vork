General

Application base URL and system-wide defaults.

Application Base URL

The public-facing base URL of this Vork instance (e.g. https://my.vork.app). Used to generate authorization form links for out-of-band approvals.
Leave blank to route through the public zero-knowledge relay at relay.vork.sh — no firewall changes required.
Set to your appliance URL to use the relay built directly into Vork. The appliance hosts the same encrypted relay protocol; you must open HTTPS to it from the internet. See vork-relay for the full zero-knowledge specification.

Default OOB Authorization Timeout

How long the relay authorization link remains valid for Telegram and interactive out-of-band requests. Background scheduled jobs use their own per-job timeout. Default: 15 minutes.
