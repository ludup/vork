General

Application base URL and system-wide defaults.

Application Base URL

The public-facing base URL of this Vork instance (e.g. https://my.vork.app). Used to generate authorization form links for out-of-band input. Leave blank to use the secure Vork Relay relay.vork.sh (see Vork Relay for full specification and code).

Default OOB Authorization Timeout

How long the relay authorization link remains valid for Telegram and interactive out-of-band requests. Background scheduled jobs use their own per-job timeout. Default: 15 minutes.
